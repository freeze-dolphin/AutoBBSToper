# AutoBBSToper

A Selenium script that top your server thread.

## Preparation

- Please install `Xvfb` to your system first:

    ```shell
    $ sudo apt install xvfb
    ```

- Then to pip:

    ```shell
    $ pip install xvfbwrapper
    ```

- Also the `Selenium`:

    ```shell
    $ pip install selenium
    ```
